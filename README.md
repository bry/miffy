## eGram
A super simple, interactive, Miffy eGram video game template built using [Gosu](http://www.libgosu.org/).

  programmed by Bryan and designed by Vouy

## Usage

### Dependencies 
- Ruby 2.6.3
- [Gosu](http://www.libgosu.org/)

### Install 

    bundle install

### Usage

    ruby gift.rb
